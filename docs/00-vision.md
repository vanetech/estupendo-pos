# Estupendo POS

Estupendo POS es un sistema web de punto de venta (POS) orientado a pequeños negocios,
iniciando con restaurantes y pizzerías.

El objetivo es permitir un control simple y claro de:
- pedidos
- apertura y cierre de caja
- gastos operativos
- reportes de ingresos y ganancias

El sistema se construye como una aplicación web con API desde el inicio,
pensada para crecer a futuro (multi-negocio, nuevos frontends, app móvil).
